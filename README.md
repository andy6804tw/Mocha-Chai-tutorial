## Getting Started
### Clone Project
 you can create a new project based on Mocha-Chai-tutorial by doing the following:

```bash
$ git clone https://github.com/andy6804tw/Mocha-Chai-tutorial.git
$ cd Mocha-Chai-tutorial
```

### Installation
When that's done, install the project dependencies.You can use npm or yarn(recommended) for dependency managementã€‚

```bash
$ npm install
```

### Running the Project

After completing the [installation](#installation) step, you're ready to start the project!

| script | Description |
| ------| ------ |
| start | Serves your app at localhost:3000 |
| test | Runs unit tests with mocha  |
